# Graph-Based Spatial Navigation & Path Planning Simulator (C++)

A C++ application that models a navigable spatial environment using graph data structures and path-planning algorithms. Developed as a Data Structures project, the system simulates navigation through a constrained environment while performing route optimization, obstacle avoidance, and shortest-path computation.

The project demonstrates core software engineering principles including object-oriented design, graph modeling, algorithm development, file persistence, and modular application architecture. It showcases concepts commonly used in CAD systems, navigation engines, geometric computing, and simulation software.

## Key Highlights

* Graph-based spatial environment modeling

## Technical Features

### Spatial Modeling

* Represented a navigable environment as a connected graph structure.
* Modeled spatial relationships between nodes using adjacency matrices.
* Implemented environment constraints through obstacle and boundary management.

### Path Planning & Algorithms

* Developed shortest-path computation using Dijkstra's Algorithm.
* Validated connectivity between nodes and ensured route feasibility.
* Simulated automated navigation through optimal path generation.

### Application Development

* Designed and implemented a modular object-oriented architecture.
* Integrated multiple system components including navigation, scoring, persistence, and user interaction modules.
* Developed an interactive menu-driven application workflow.

### Data Management

* Persisted user scores using file-based storage.
* Implemented leaderboard functionality using Binary Search Trees.
* Managed runtime state using STL containers and dynamic memory allocation.
